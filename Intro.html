Source code Description

File Place File Name Description Functions
Root Intro.html Introduction of each source code file.
Root index.html Manu bar, including the logo, manu buttons, and search box. Use iframe to change the main content. Used functions mainly in main.js and laptop.js.
buildLaptopsDics() - Build three laptop dictionaries, AmazonLaptopsInfoDic, BestbuyLaptopsInfoDic,and EbayLaptopsInfoDic.
enableSearch() - Enable searching within title, e.g. "Asus HP".
Root main.html Show in main content area, including news slider and deals
Root product_list.html Show in main content area, including filter and grid laptop info. Used functions mainly in filter.js and laptop.js.
Root product.html Show in main content area, including reviews, specifications, prices, shares and so on. Used functions mainly in product.js and laptop.js.
Root compare.html Show in main content area, including radar chart, polar charts, specifications, shares and so on. Used functions mainly in compare.js and laptop.js.
js main.js Build the laptop metadata dictionaries, control page changing in iframe and enable searching. buildLaptopsDics(), changePage(url), changeToCompare(addr, titlelist), returnMetadata(metadataOriginal, t), checkDataCondition(), makeDic(), getMetadata(url, callback), doJSONPCall(jsonpURL), removeByValue(arr, val), outputLaptop(title), search(), enableSearch()
js laptop.js Initialize laptopQueue to show every laptop in dictionaries. Count how many laptops have been selected for comparison. setup(), showSearchResults(), showLaptopsInQueue(laptopMetadata), hangeToCompare(addr, selectLaptopsTitle), selectedPercentagePieChart(selectLaptops)
js filter.js Using with laptop.js to support the filtering of price, weight, screen size etc. Also support "sortBy" function. priceFilter(min, max), manufacturerFilter(), weightFilter(min, max), screenSizeFilter(min, max), OSFilter(), sortLaptops(event), sortLaptopsBy(sortBy)
js product.js Used for building product.html page. getLaptopInfo(urlA, urlB, urlE), metaDataFromThreeSources(title), show(laptopMetadataAmazon, laptopMetadataBestbuy, laptopMetadataEbay), showSpecifications(laptop, laptopMetadataAmazon), showAllReviews(laptop, laptopMetadataAmazon), findReview(label, reviewList)
js compare.js Used for building compare.html page. buildList(titleList), getSelectLaptopInfo(titleList), getSelectLaptopInfo(titleList), drawPolarArea(titleList), drawStars(laptopMetadataAmazon, div), drawSpecifications(titleList, width)
js Chart.js Used for drawing radar chart and polar chart. Chart(ctx).PolarArea(data, option), Chart(ctx).Radar(data, option)
css css files Control the showing style of each elements in pages.
data data files Pics and data used in this website.
docs documentation files Store pdf files.
resources jQuery relevent files jQuery relevent .js and .css files.
engine1 news slider files Relevent .js, .css and pics used in the news slider.

Comments

1. Because the item matching probelm across three sources, only use 11 different laptop models in this prototype.
2. Now the links in laptop drop down only link to the product_list.html without filter.
3. Each subsection of filter bar in product_list page works seperately.
4. In product.html, it tries to use star scores and the first sentence to summurize good and bad reviews, but it is not very accurate based only on the first sentence in review. Can use semantic analysis to increase accracy in future.
